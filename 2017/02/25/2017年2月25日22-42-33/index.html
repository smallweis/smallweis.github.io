博客的搭建——初识hexo

发表于 2017-02-25 | 分类于 随手小笔 | | 阅读次数

文本摘要：用hexo+next搭建了一个关于技术的小博客，本文记录了搭建的过程和心得

  一直有建立建立博客的打算，但是目前流行的博客开源框架搭建比较复杂。如PHP比较优秀的开源博客WordPress，本人表示对php一窍不通所以搭建博客这事就一直搁浅了。搭建博客也是对自己技术和生活的一个更好记录吧!嘻嘻,最近发现一款较好的开源博客hexo+next，它的搭建比较特殊，是通过本地的预编译把动态文件生成静态静态网页上传到服务器——本博客上传到了gitHub。

gitHub稳定的免费空间用来做博客再好不过了！而且很多主题文件简洁大方,让我们看看这款神奇的hexo+next。
